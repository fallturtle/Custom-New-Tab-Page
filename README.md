# Custom-New-Tab-Page
GitHub page for a custom new tab project I have been working on.
You must download the folder and then open the new tab file. Then use a redirect tool such as [Custom New Tab]([url](https://chromewebstore.google.com/detail/custom-new-tab/lfjnnkckddkopjfgmbcpdiolnmfobflj)) and set it to the url of that file. The other file in the folder is for the games that can be toggled on in the settings found in the side pannel.
There is also a reddit post for this project.
P.S. If someone with chrome web store developer wants to modify this to replace the actual chrome new tab instead of dedirecting by all means do so and then please inform me, I would love to replace my chrome new tab instead of redirecting it!
